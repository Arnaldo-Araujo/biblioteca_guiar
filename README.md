## Scripts Úteis

Este projeto contém scripts utilitários na raiz do repositório para facilitar o desenvolvimento:

- `./run_on_device.sh`: Executa o app automaticamente no primeiro dispositivo encontrado conectado. Útil para iniciar o app rapidamente no celular conectado por USB.
- `./build_aab_obfuscated.sh`: Gera o build Android em formato App Bundle (`.aab`) aplicando ofuscação no código Dart para proteção do aplicativo antes do deploy na Google Play.
